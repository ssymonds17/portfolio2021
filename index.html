The application allows users to move around several different base maps, select countries and view different data points for the country selected, including covid-19, news, national holiday, and earthquake data, as well as weather information for individual cities.

company directory

Company Directory is a HR-style application that allows the user to manage employee, department and location records for a particular company via CRUD operations and also provides employee search functionality via filters on employee names, or via department or location.

The application was developed using React on the frontend and CRUD operations are carried out via api requests to an independent api. This api utilises MySQL and is accessed using PHP.

basketball fantasy draft

Basketball fantasy draft application that replicates the functionality of ESPN's famous fantasy draft game.

Developed exlusively with React, the application allows a user to compete against nine AI competitors to draft a basketball squad from a list of NBA legends. After assembling their squad, the user will match up against each of their rivals in an attempt to go undefeated. The user's score is then saved for future reference.
